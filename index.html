About Me

I am a graduate research assistant pursuing my M.S. in Computer Science at Missouri State University, where I focus on Human-Computer Interaction and Human-AI Collaboration. My path into computing grew from an undergraduate foundation in Energy Science and Engineering at KUET, Bangladesh, where I developed a deep appreciation for simulation, data analysis, and systems thinking. I love digging into complex problems, whether that means characterizing memory behavior in LLM inference systems or modeling how materials can make buildings more energy-efficient. I am always curious, always learning, and happiest when I get to build something that solves a real challenge.

Research Experience

2026

Tail Latency Characterization Under Concurrent LLM Inference

Examined how response times in a large language model server degrade as more users send requests simultaneously, especially with very long inputs. Designed a testing framework that captured latency across different concurrency levels, revealing that the slowest responses get hit disproportionately harder as contention increases.

2026

KV Cache Memory Scaling in LLM Inference Systems

Investigated how GPU memory usage grows in large language model servers as inputs get longer and more users send requests at the same time. Tracked KV cache usage live through the server's own metrics to understand where memory pressure builds up under real-world conditions.

2025

GPU-CPU Memory Transfer Latency Benchmarking

Evaluated data transfer efficiency between CPU and GPU by testing multiple buffer sizes on a production GPU server. Developed a test setup that measured latency in both directions across thousands of iterations, comparing performance when idle and under active CPU load.

2023 | Undergraduate Thesis

Thermal Performance of Buildings Integrated with Phase Change Materials

Explored how phase change materials could help keep buildings cooler without active cooling systems. Used DesignBuilder and EnergyPlus to simulate their performance when embedded into roofs of single-story buildings across different climate zones in Bangladesh, measuring energy savings and thermal comfort improvements.

Projects

2022

Automated Single Axis Solar Tracking Based Battery Charging System

Built a solar tracking system using an Arduino microcontroller and C++ that automatically adjusts a solar panel's angle to follow the sun throughout the day. Ran a side-by-side comparison with a standard static panel to measure the efficiency gains, and designed an integrated battery storage setup to optimize how the harvested energy was stored and used.
